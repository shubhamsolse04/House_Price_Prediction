# House Price Prediction

An end-to-end Machine Learning project that predicts house prices based on various house characteristics.

## Live Demo

https://housepricepredictionmachinelearningmodel.streamlit.app/

## Features

- Predict house prices using Machine Learning
- Interactive Streamlit web application
- Uses 13 house attributes
- Displays price in Rupees, Lakhs and Crores

## Dataset Features

- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status

## Machine Learning Algorithms Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor
- XGBoost Regressor

## Best Model

Gradient Boosting Regressor

R² Score: 0.666

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Streamlit
